# generand
```
USAGE:
    generand [OPTIONS] [DICTIONARY]

ARGS:
    <DICTIONARY>

OPTIONS:
    -c, --cats
    -d, --digits
    -f, --full
    -h, --hex
        --help               Print help information
    -n, --number <NUMBER>    [default: 1]
    -s, --size <SIZE>        [default: 12]
    -V, --version            Print version information
   ```
